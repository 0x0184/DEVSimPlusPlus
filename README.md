## Build
### 1. Windows
```ps1
>> cmake CMakeLists.txt
>> make
```
### 2. MacOS
```zsh
$ brew install cmake
$ cmake CMakeLists.txt
$ make
```
### 3. Linux
```bash
$ apt install cmake
$ cmake CMakeLists.txt
$ make
```
